Campus-Viewer
=============

Mit der Campus Viewer App sollen Menschen, 
die noch nie auf dem Campus der Hochschule Fulda waren, 
die Möglichkeit erhalten, 
sich einen Überblick über den Campus der Hochschule zu verschaffen.
